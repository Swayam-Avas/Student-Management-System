#  Student Management System

A simple **Java console-based application** to manage student records using CRUD operations.

---

## 🚀 Features

* ➕ Add Student
* 📋 View Students
* 🔍 Search Student by ID
* ✏️ Update Student Details
* ❌ Delete Student

---

## 🛠️ Technologies Used

* Java
* OOP Concepts
* ArrayList (Data Structure)
* VS Code

---

## 📂 Project Structure

```
Student-Management-System/
│── Main.java
│── Student.java
```

---

## ▶️ How to Run

1. Compile the program:

```
javac Main.java Student.java
```

2. Run the program:

```
java Main
```

---

## 💡 Sample Output

```
1. Add Student
2. View Students
3. Search Student
4. Delete Student
5. Update Student
6. Exit
```
